## 📝 Descrição
Calculadora desenvolvida como projeto pessoal para praticar lógica de programação, operações matemáticas e criação de interfaces simples de forma funcional e intuitiva.

## 🚀 Funcionalidades
- Operações matemáticas básicas (Soma, Subtração, Multiplicação e Divisão).
- Interface responsiva e de fácil interação.
- Limpeza de dados (Botão C/CE) e tratamento de erros (ex: divisão por zero).

## 🛠️ Tecnologias Utilizadas
- **JavaScript (ES6+)** - Lógica das operações.
- **HTML5** - Estruturação da interface.
- **CSS3** - Design e estilização visual.

## 💻 Como Utilizar
1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em seu navegador para utilizar a calculadora.
